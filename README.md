# ABOUT

华中科技大学的一些报告/实验/课设（子文件夹下有些是报告，有些是实验，有些是课设，有些几个都有）。

每个子文件夹都是一个项目。项目的`.git/`文件夹都已被移除。也就是说没有历史版本。

标记(x)的项目是我认为过于无意义的项目。

基于CC BY-NC发布（若子文件夹下有LICENSE文件则按子文件夹下的LICENSE发布，否则CC BY-NC发布）。
